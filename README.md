# obuv-shop-project

Этот репозиторий содержит проект интернет-магазина обуви, разработанный в рамках учебного задания. Проект включает в себя три модуля, каждый из которых представляет собой отдельную версию или этап разработки магазина.

## Структура проекта

- **модуль 1** - Первоначальная версия интернет-магазина обуви
  - Содержит основной функционал: авторизация, просмотр товаров, оформление заказов
  - Включает SQL-скрипт для базы данных

- **модуль 2** - Улучшенная версия с дополнительным функционалом
  - Добавлены HTML-алгоритм и стили
  - Улучшенный интерфейс и пользовательский опыт

- **модуль 3** - Финальная версия с расширенным функционалом
  - Добавлены возможности администратора
  - Расширенные функции управления пользователями

## Установка и запуск

1. Клонируйте репозиторий
2. Установите зависимости (если применимо)
3. Настройте базу данных, используя предоставленные SQL-скрипты
4. Запустите локальный сервер (например, Apache с PHP)
