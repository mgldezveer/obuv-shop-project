# Информационная система "ООО Обувь"

## Описание проекта
Система управления магазином обуви с ролевой моделью доступа.

## Технологический стек
- **Backend:** PHP 8.2.12
- **Database:** MySQL
- **Frontend:** HTML5, CSS3, JavaScript
- **Server:** Apache (MAMP/XAMPP)

## Структура проекта
- `admin_orders.php` - страница управления заказами для администратора
- `admin_products.php` - страница управления товарами для администратора
- `config.php` - конфигурационный файл подключения к базе данных
- `footer.php` - нижняя часть страницы (шаблон)
- `header.php` - верхняя часть страницы (шаблон)
- `login.php` - страница входа в систему
- `logout.php` - страница выхода из системы
- `orders.php` - страница просмотра заказов для пользователя
- `products.php` - страница просмотра товаров
- `style.css` - файл стилей
- `algorithm.html` - алгоритм работы системы
- `images/` - директория с изображениями (логотипы, товары и т.д.)
  - `favicon.ico` - иконка сайта
 - `logo.png` - логотип компании
  - `4.jpg`, `5.jpg`, `8.jpg`, `9.jpg` - изображения товаров
